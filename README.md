# Requirements

- Neovim >= 0.10
- NodeJs
- GCC
- Ripgrep
- LazyGit

## GCC
`brew install gcc`

## NodeJS
`brew install node`

## Neovim
`brew install neovim`

## Ripgrep
`brew install ripgrep`

## Lazygit
`brew install lazygit`

# Locations
Estas configuraciones deben ir en:

## Windows
`C:Users/AppData/Local/nvim`

## Linux: 
`~/.config/nvim`
